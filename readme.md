## manager 模块

### 启动
1. 不带参数启动时，默认的是设置自己为 leader节点
```$xslt
java -jar cluster-mq-manager.jar
```

2. 带参数启动节点
```$xslt
java -jar cluster-mq-manager.jar --cluster.ip=ip1 --cluster.port=port1 --server.port=8080
```
